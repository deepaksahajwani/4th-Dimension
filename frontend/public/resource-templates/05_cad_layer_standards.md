## Best Practices

1. **Never use Layer 0** for drawing objects
2. **Always freeze unused layers** rather than turning off
3. **Lock reference layers** to prevent accidental edits
4. **Purge unused layers** before issue
5. **Use layer filters** for efficient layer management
