# Brand Guidelines
## 4th Dimension Architects

---

## 1. Logo

### Primary Logo
The 4th Dimension logo consists of:
- The "4D" symbol in the company blue
- "4th Dimension" wordmark
- "Architects" tagline

### Logo Variations
| Version | Use Case |
|---------|----------|
| Full color | Primary use on white backgrounds |
| Reversed (white) | On dark backgrounds |
| Single color (black) | Documents, fax |
| Icon only (4D) | Social media, favicon |

### Clear Space
- Minimum clear space = height of "4" in logo
- No other elements within this zone

### Minimum Size
- Print: 25mm width
- Digital: 120px width

### Don'ts
- ❌ Don't stretch or distort
- ❌ Don't change colors
- ❌ Don't add effects (shadows, outlines)
- ❌ Don't place on busy backgrounds
- ❌ Don't rotate the logo

## 3. Typography

### Primary Font: Montserrat
- **Headings**: Montserrat Bold
- **Subheadings**: Montserrat SemiBold
- **Body**: Montserrat Regular
- **Captions**: Montserrat Light

### Secondary Font: Open Sans
- Use for long-form documents
- Better readability at small sizes

### Font Sizes (Print)
| Element | Size | Weight |
|---------|------|--------|
| Main Title | 24-36pt | Bold |
| Section Heading | 18pt | SemiBold |
| Subheading | 14pt | SemiBold |
| Body Text | 11pt | Regular |
| Caption | 9pt | Light |

### Font Sizes (Digital)
| Element | Size |
|---------|------|
| H1 | 32-48px |
| H2 | 24-32px |
| H3 | 18-24px |
| Body | 16px |
| Small | 14px |

## 4. Stationery

### Business Card
- Size: 90mm x 55mm
- Paper: 350gsm matte
- Front: Logo, name, title
- Back: Contact details

### Letterhead
- A4 size
- Logo top-left
- Contact details bottom
- Margin: 25mm all sides

## 5. Presentations

### PowerPoint Template
- Title slide: Large image + logo
- Content slides: Clean layout, 4D colors
- Section dividers: Full-bleed images
- Charts: Use brand colors only

### Slide Guidelines
- Maximum 6 bullet points per slide
- Font minimum: 24pt for body
- High-quality images only (min 1920x1080)
- Use master slides, don't create custom layouts
